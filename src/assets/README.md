# Convenciones de assets

- `styles/`: hoja global y estilos compartidos de la UI.
- `images/`: imagenes estaticas del producto (logos, placeholders, etc.).
- `fonts/`: fuentes locales cuando aplique licencia/uso offline.
- `runtime/`: recursos auxiliares usados en ejecucion (si se incorporan en futuras tareas).

Regla: priorizar rutas claras y separar recursos por tipo para evitar mezclar estilos, media e iconografia en una sola carpeta.
